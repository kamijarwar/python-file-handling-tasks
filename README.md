# Python File Handling Tasks 🐍

A collection of beginner-friendly Python file handling tasks designed to practice core programming concepts such as file reading, writing, searching, and data processing.

---

## 📌 About the Project

This project includes **15 Python tasks** that help strengthen understanding of file handling in Python. Each task focuses on a different concept such as:

- Reading and writing files  
- Searching text in files  
- Counting words and lines  
- Working with student records  
- String manipulation in files  

---

## 🚀 Features

- Beginner-friendly Python code  
- Practical file handling examples  
- Clean function-based structure  
- Easy to understand and modify  

---
Go to project folder:

cd python-file-handling-tasks
Run any task:
python task1.py

Purpose

This project is created for practice and learning purposes to improve Python file handling skills and build a strong programming foundation.
